# expres-starter

this is my personal express starter with mvc and 
this project have session and token auth



## Start Server

```
npm run start  //or
npm start     //or
nodemon app.js  //or
```



## Start Laravel Mix

```
npm run dev      //or
npm run watch    //or

```



## Update and Install Packages

```
npm-upgrade
npm install
```



## access parts

```
http://localhost:3000/signup  // user register
http://localhost:3000/signin  // user login
```
